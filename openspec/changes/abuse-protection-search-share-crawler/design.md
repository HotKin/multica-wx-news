# abuse-protection-search-share-crawler Design

## 搜索

- query 长度限制。
- 控制字符和危险 payload 拦截。
- 匿名限流。
- 不返回未发布、已下架、投诉冻结内容。

## 分享

- 分享参数白名单。
- 禁止开放重定向。
- 分享标题、摘要、专题名和来源名全部按文本输出，不信任 HTML。

## 采集

- 仅允许来源配置中的域名和路径。
- 频率限制、重试上限、指数退避、熔断。
- 禁用开关可停止某来源或全部 worker。
- 代理凭证和第三方 key 只在服务端使用。

## 外链媒体

- 禁止访问内网、link-local、metadata IP。
- 图片/视频需要 MIME、大小和超时限制。
- 必要时重编码或使用安全代理。
