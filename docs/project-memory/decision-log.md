## 废弃或被修订的历史假设

- “前台完全不做账号、收藏仅本地”已被修订为“微信登录入口先不实现，但账号/收藏接口边界保留”。
- “无授权来源默认只摘要展示”被来源规则替代：每个来源按授权类型和字段级可展示配置执行，默认值仍不得自动允许全文。
- “仓库不可 checkout”已修正；当前 workspace 受管仓库为 `HotKin/multica-wx-news`，`HotKin/news-app` 未配置在本 workspace。

## 发布门禁

进入 staging 前：

1. 工程实现可运行，基础 CI 通过。
2. 数据层、状态机、RBAC、审计和下架清理有测试覆盖。
3. 安全复核无 P0。
4. 运行环境变量、健康检查、回滚步骤明确。

真实内容上线前：

1. 首批来源清单、授权材料、投诉联系人和紧急责任人确认。
2. 生产统计/错误上报供应商与字段/保留周期确认。
3. 下架/投诉演练通过。
4. 真实应用验收通过。
5. 风险接受记录和回滚预案归档。
