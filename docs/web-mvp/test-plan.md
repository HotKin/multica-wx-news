# 快闻 Web MVP 验收计划与测试矩阵

## 测试结论

KIN-13 已完成测试计划与用例矩阵设计。该结论仅代表测试计划完整，不代表产品实现验收通过。

## P0 测试矩阵

| 模块 | 必测点 |
| --- | --- |
| 首页 | 推荐、最新、频道入口、热榜、专题入口；不展示未发布、待审、驳回、下架或投诉冻结内容。 |
| 频道 | 只展示该频道已发布内容；无内容空状态；停用频道隐藏。 |
| 详情 | 标题、分类标签、文章内容、作者、来源、发布时间；未发布/下架/投诉冻结不可读。 |
| 热榜 | 只展示已发布未下架内容/专题；第三方热榜缺失降级；人工干预审计。 |
| 专题 | 发布专题展示简介和文章列表；关联新闻下架后自动隐藏。 |
| 搜索 | 只返回已发布未下架内容；无结果、非法输入、服务失败降级。 |
| 收藏/账号接口 | 未授权失败；登录后新增、取消、列表查询；下架后收藏打开不可读。 |
| 分享 | 公开链接可打开；分享标题和参数净化；不泄露后台字段。 |
| 后台运营 | 来源、文章、频道、推荐位、热榜、专题、审核、发布、下架、投诉、数据看板。 |
| 来源合规 | 缺必填不能启用；字段级授权严格执行；授权到期预警/暂停。 |
| 审核发布 | 自动审核、人工审核、立即/定时/批量发布最终校验。 |
| 投诉下架 | 5 分钟详情不可读，30 分钟搜索/热榜/专题/推荐位/收藏/cache/sitemap 清理。 |
| 埋点指标 | 白名单事件、指标口径、敏感字段禁入。 |
| 安全滥用 | XSS、open redirect、SSRF、搜索限流、采集熔断、密钥隔离。 |
| 兼容回归 | 桌面和移动 H5；核心页面文本、按钮、列表不重叠。 |

## 执行门槛

真实应用验收需要：

1. 工程输出可运行实现、运行命令、测试数据和已知限制。
2. 如需真实内容验收，业务/法务确认来源、授权和投诉联系人。
3. 安全复核无 P0 阻断项。
4. CI 基础门禁可运行。

## 输出要求

- 报告实际运行命令、环境、版本和数据集。
- 区分通过、失败、无法验证和阻塞项。
- 对失败项提供复现步骤。
- 给出是否可进入 staging 的结论。
